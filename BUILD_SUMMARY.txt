📋 GETTING STARTED

1. Installation
   cd frontend
   npm install

2. Development
   npm run dev
   → Runs on http://localhost:3000

3. Production Build
   npm run build
   → Creates optimized dist/ folder

4. Backend Setup
   cd backend
   npm install
   npm run dev
   → Runs on http://localhost:3001

5. Environment Variables
   Copy .env.example to .env
   Add your API keys:
   ├─ ODDS_API_KEY
   └─ SPORTSDATA_IO_API_KEY
