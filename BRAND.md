## Visual Identity

### Design Philosophy

**Less:** Sportsbook, Entertainment, Noise, Hype
**More:** Trading terminal, Intelligence, Data, Precision

### Primary Colors

```
Market Black:    #0F1115  (Primary background — professional, confident)
Terminal Navy:   #131A24  (Secondary surfaces — depth)
Signal Green:    #00D27A  (Alerts, CTAs, positive signals)
Data Blue:       #2B7FFF  (Charts, data visualization)
Slate:           #6B7280  (Secondary text, hierarchy)
Cloud:           #E5E7EB  (Light backgrounds)
White:           #FFFFFF  (Text, contrast)
```

### Typography

**Primary Font Stack:**
```
font-family: "Inter", "Geist", "SF Pro", sans-serif;
```

**Monospace (for data/numbers):**
```
font-family: "JetBrains Mono", monospace;
```

### Design Principles

1. **Minimal, clean interface** — No unnecessary elements
2. **Data-forward** — Show information, not hype
3. **Professional aesthetic** — Bloomberg Terminal, not DraftKings
4. **Dark mode by default** — Terminal-like appearance
5. **Responsive design** — Works on desktop, tablet, mobile

## Component Guidelines

### Buttons

**Primary Button (Signal Green)**
- Use for main CTAs
- Examples: "Execute Trade", "Save Position", "Analyze Signal"

**Secondary Button (Terminal Navy + Border)**
- Use for alternative actions
- Examples: "View Details", "Export", "Share"

**Ghost Button (Transparent)**
- Use for tertiary actions
- Examples: "Dismiss", "Cancel", "Learn More"

### Cards

- Subtle border (1px, Slate @ 10%)
- Light hover effect (border becomes Signal Green)
- No drop shadows — keep minimal and clean

### Forms

- Dark backgrounds (Terminal Navy)
- Signal Green focus state
- Clear labels above inputs
- No red error states — use warning amber

### Data Tables

- Striped rows (hover = subtle Signal Green tint)
- Monospace for numbers
- Right-align numerical data
- Left-align text data

## Writing Guidelines

### Headlines

✅ "Market Movement Over the Past 24 Hours"
✅ "Your Portfolio's Closing Line Value"
✅ "Compare Odds Across Sportsbooks"

❌ "Amazing New Signals!"
❌ "These Picks Are 🔥"
❌ "Beat Vegas Today!"

### Body Copy

- Short sentences
- Active voice
- Data-forward language
- Avoid gambling jargon and superlatives

### Key Terms (USE)
Market, Price, Signal, Intelligence, Probability, Value, Edge, Position, Tracking, Closing Line, Execution, Movement, Liquidity, Performance, Portfolio, Terminal, Algorithm, Model, Probability, Analysis

### Terms to Avoid
Locks, Guaranteed, Winners, Easy Money, Max Bet, Can't Miss, Betting Guru, Whale, Hammer, Hot Pick, Lock of the Day, Betting Tips

## Do's and Don'ts

### ✅ DO

- Be analytical and precise
- Use data to support claims
- Reference institutional markets
- Build credibility through transparency
- Show your work (methodology, models, reasoning)
- Use professional language
- Focus on process and discipline
- Emphasize long-term thinking

### ❌ DON'T

- Make guaranteed predictions
- Use hype or urgency tactics
- Reference gambling influencers
- Promise easy money
- Use bright colors or loud design
- Sacrifice clarity for style
- Use gambling terminology
- Appeal to emotions

## Brand Assets

### Logo Usage
- Primary: M9 Terminal wordmark
- Secondary: Oddsify Labs logo
- Never distort or change colors
- Maintain minimum clearance around logo

### Color Usage
- Signal Green for actionable items
- Data Blue for informational content
- Slate for secondary information
- Ensure WCAG AA contrast compliance

### Typography
- Inter for UI and headlines
- JetBrains Mono for code and numbers
- Maintain clear hierarchy
- Use weight and size for emphasis, not all caps

---

## Accessibility

- All colors meet WCAG AA contrast minimums
- Text size minimum 14px
- Focus states clearly visible
- No color-only information
- Monospace for numbers (easier to scan)
- Alt text for all images
- Keyboard navigation support

## Tone by Context

### Error Message
Clear, helpful, non-judgmental.

Good: "Connection lost. Reconnecting... Offline since 2:15 PM."
Bad: "OH NO! SOMETHING WENT WRONG!"

### Success Message
Understated, factual.

Good: "Position saved. Signal sent to your alerts."
Bad: "🎉 YOU DID IT! 🎉"

### Empty State
Helpful, actionable.

Good: "No active signals yet. Configure your filters to start tracking."
Bad: "Nothing here!"
